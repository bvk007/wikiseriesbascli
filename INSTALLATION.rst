============
Installation
============

At the command line::

    $ pip install wikiseriesbascli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv wikiseriesbascli
    $ pip install wikiseriesbascli

Or, if you are using pipenv::

    $ pipenv install wikiseriesbascli
Or, if you are using pipx::

    $ pipx install wikiseriesbascli
